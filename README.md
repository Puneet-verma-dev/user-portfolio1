Personal Portfolio Website  
A modern, fully responsive, dark-themed portfolio website showcasing my work, skills, and interactive projects. Built with **HTML**, **CSS**, and **JavaScript**, this project demonstrates clean UI/UX principles, animation workflows, reusable JavaScript components, and modular file design.  

This website is created to represent my identity as a **Full-Stack Web Developer** and serves as my public-facing professional profile.

# 📌 Overview  
This portfolio is designed with a strong focus on:

- **Professional identity branding**
- **Strong UI/UX consistency**
- **Custom animations without external libraries**
- **Performance and responsiveness**
- **Showcasing real working mini-projects**

The website contains multiple interactive components such as a **To-Do App**, **Calculator Demo**, **Modals**, and **Animated Progress Bars**, all built from scratch.

# 🎨 Features (Detailed)

### 🔥 1. Animated Hero Section  
- Large dynamic typography  
- Background watermark “PV”  
- Animated glow effects  
- CTA buttons  
- Availability pulse indicator  

### 📡 2. Scroll & Fade Animations  
- Intersection Observer API  
- Smooth transitions  
- Lazy reveal for sections  

### 🧰 3. Fully Working Project Demos  
**To-Do App (Mini Version)**  
- Add tasks  
- Mark as completed  
- Delete tasks  
- Live list update  
- Local state  

**Calculator (Mini Version)**  
- Arithmetic operations  
- Chained operations  
- Decimal handling  
- Keyboard support

- HTML separated  
- CSS modular & theme-based  
- JS event-driven  
- Reusable components (modal, calculators, tasks, animations)  

### 🎛 5. Responsive Layout  
- Mobile-first approach  
- Tablet optimization  
- Desktop widescreen enhancements  

### 🧲 6. Animated Skill Bars  
- Auto-fill bars when visible  
- Per-skill animation speed and width  

### 🪄 7. Infinite Scrolling Ticker  
- CSS-based marquee  
- Loops smoothly  
- Lightweight animation  

### 🪟 8. Modal System  
- Injects custom HTML  
- Smooth fade animation  
- ESC-key support  
- Auto-init scripts per modal

# ⚙️ Installation

You don’t need Node.js or frameworks. This project is 100% static.

### **Step 1 – Download ZIP**  
Unzip into a folder.

### **Step 2 – Open the file**  
Just open:


index.html


That’s it. The site runs offline and requires no server.

# 🌍 Deployment Guide

### **Deploy on GitHub Pages**
1. Create a GitHub repo  
2. Upload these files  
3. Go to → Settings → Pages  
4. Select branch: `main`  
5. Save  

Your site will appear at:  
`https://yourusername.github.io/your-repo-name/`

# 🏗️ Technical Architecture

### **CSS Architecture**
- Global variables (`:root`)  
- Reset styling  
- Component-based sections  
- Media queries at:  
  - 1100px  
  - 768px  
  - 480px  

### **JavaScript Architecture**
Modules inside `script.js`:

| Module | Purpose |
|--------|---------|
| Navbar Scroll Handler | Shrink animation |
| Fade-In System | IntersectionObserver |
| Skill Bar Engine | Auto-width animator |
| Mini To-Do App | Functional demo list |
| Mini Calculator | Full calculator engine |
| Modal Engine | Reusable modal logic |
| Project Demo Loader | Injects content dynamically |
